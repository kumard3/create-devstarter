
# `create-devstarter`

🏗 Personal CLI to bootstrap new starters from the given templates. added yarn support

## Installation & Usage

```bash
npm init create-devstarter
# or
npx create-devstarter
# or
npm install -g create-devstarter
create-project
# or
yarn create devstarter
# or
create-devstarter

```
